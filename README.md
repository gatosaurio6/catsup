Comandos para SUBIR tus cambios locales a GitHub

1) git status

2) git add .      <----    # selecciona todos los archivos para modificar

Si solo quieres agregar un archivo específico, cambia el punto por el nombre del archivo
2)  git add settings.py <---- (ejemplo)


3) git commit -m "Escribe aquí un resumen corto de lo que cambiaste"    <------ #escribir el cambio 

4) git push origin main  <------ #sube los archivos

1) git status    <----- #Asegúrate de que no tienes trabajo a medias sin guardar


2) git pull origin main   <----- #Trae y fusiona todo lo nuevo de la nube a tu PC:
